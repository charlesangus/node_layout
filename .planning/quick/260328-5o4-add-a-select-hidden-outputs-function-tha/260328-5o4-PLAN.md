1. Get all currently selected nodes into a set (using `nuke.selectedNodes()`). If none selected, return early (optionally show `nuke.message`).
2. Build an id-based set of the selected nodes for fast lookup: `selected_ids = {id(n) for n in selected_nodes}`.
3. Iterate over all nodes in the current group context (`nuke.allNodes()`). For each candidate node:
   - Check if it has `hide_input` knob set to True (same pattern as `_hides_inputs` in node_layout.py: `knob = candidate.knob('hide_input'); knob is not None and knob.getValue()`).
   - If yes, check all its inputs (`candidate.input(i)` for `i in range(candidate.inputs())`). If any input's `id()` is in `selected_ids`, this candidate is a hidden output.
4. Collect all hidden output nodes.
5. Do NOT clear existing selection — ADD the hidden output nodes to the current selection by setting `n["selected"].setValue(True)` for each found node. This way the original selection is preserved and the hidden outputs are added.
