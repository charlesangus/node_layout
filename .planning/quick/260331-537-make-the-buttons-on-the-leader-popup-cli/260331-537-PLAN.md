Add a `dispatch_key(key_letter)` top-level public function after the `_disarm()` function. This function accepts a single uppercase letter string (e.g. "W", "V") and dispatches it through the same logic as eventFilter's KeyPress branch:

1. Build a letter-to-Qt-key mapping: `_LETTER_TO_QT_KEY = {"V": Qt.Key.Key_V, "Z": Qt.Key.Key_Z, ...}` for all 10 keys. Place this as a module-level dict near the dispatch tables.

2. `dispatch_key(key_letter)` implementation:
   - Look up the Qt key from `_LETTER_TO_QT_KEY[key_letter]`

   - Check `_CHAINING_DISPATCH_TABLE` second (chaining): if found, hide `_overlay` if not None, then call the dispatch function (same as eventFilter chaining branch)
