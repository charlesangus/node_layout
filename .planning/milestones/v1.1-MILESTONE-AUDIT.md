### HORIZ-01 Note

The REQUIREMENTS.md text says "User can run 'Layout Upstream Horizontal'" but no standalone `Layout Upstream Horizontal` menu command exists in `menu.py`. This was an intentional post-UAT redesign decision documented in Phase 11 VERIFICATION.md (lines 68–75). The capability is delivered via:
1. `Layout Selected Horizontal` — sets mode='horizontal' on spine nodes (HORIZ-02)
2. Normal `Layout Upstream` — auto-dispatches to horizontal path via stored mode replay (HORIZ-03)

All three tracking sources (checkbox, VERIFICATION.md, SUMMARY frontmatter) agree the requirement is satisfied. Status: **satisfied** — requirements text reflects the pre-redesign command name, not the final delivered architecture.
