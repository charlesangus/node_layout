Axis conventions in horizontal mode:
- B-spine direction: LEFT (negative X from root)
- Side inputs go ABOVE (negative Y — same direction as vertical mode)
- Root is RIGHTMOST (highest X)
- Each upstream input[0] steps one node width + gap to the LEFT

Key formulas:
  step_x = int(horizontal_subtree_gap * scheme_multiplier)   # gap portion only
  cur_x decremented by: step_x + upstream_node.screenWidth()

  side_y = cur_y - vertical_gap_between(side, spine_node, ...) - side_node.screenHeight()
  side_x = _center_x(side_node.screenWidth(), cur_x, spine_node.screenWidth())

  mask kink (two-pass):
    Pass 1 (ancestor-first): cumulative_kink_y += compute_dims(mask_node)[1] + _subtree_margin(...)
    Pass 2: cur_y = spine_y + kink_y_per_index[index]

    1. Build spine_nodes list by walking input[0] chain from root: [root, root.input(0), root.input(0).input(0), ...]
    2. If memo is None, create memo = {}
    3. Resolve scheme_multiplier at entry (same pattern as existing place_subtree)
    4. step_x = int(prefs.get("horizontal_subtree_gap") * scheme_multiplier) — gap portion only

    Pass 1 (accumulate mask kink, iterate spine_nodes ancestor-first, i.e. reversed):
      cumulative_kink_y = 0
      kink_y_per_index = [0] * len(spine_nodes)
      for reverse_index in range(len(spine_nodes) - 1, -1, -1):
          kink_y_per_index[reverse_index] = cumulative_kink_y
          spine_node = spine_nodes[reverse_index]
          for slot in range(spine_node.inputs()):
              if _is_mask_input(spine_node, slot):
                  mask_node = spine_node.input(slot)
                  if mask_node is not None:
                      mask_h, mask_w = compute_dims(mask_node, memo, ...)
                      mask_margin = _subtree_margin(spine_node, slot, ...)
                      cumulative_kink_y += mask_h + mask_margin

    Pass 2 (place each spine node):
      cur_x = spine_x
      for index, spine_node in enumerate(spine_nodes):
          cur_y = spine_y + kink_y_per_index[index]
          spine_node.setXpos(cur_x)
          spine_node.setYpos(cur_y)
          # Place side inputs (slot > 0, non-mask) above spine node using vertical formulas:
          for slot in range(1, spine_node.inputs()):
              if _is_mask_input(spine_node, slot): continue  # mask handled separately
              side_node = spine_node.input(slot)
              if side_node is None: continue
              side_h, side_w = compute_dims(side_node, memo, ...)
              gap = vertical_gap_between(side_node, spine_node, snap_threshold, scheme_multiplier)
              side_x = _center_x(side_node.screenWidth(), cur_x, spine_node.screenWidth())
              side_y = cur_y - gap - side_node.screenHeight()
              side_node.setXpos(side_x)
              side_node.setYpos(side_y)
          # Advance cur_x leftward for next spine node
          next_node = spine_node.input(0) if index < len(spine_nodes) - 1 else None
          if next_node is not None:
              cur_x -= (step_x + next_node.screenWidth())

    Do NOT recurse into side input subtrees here — simple direct placement only.
