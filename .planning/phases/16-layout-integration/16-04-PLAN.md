**Change B — Gap 1: Position non-frozen upstream nodes above anchored blocks**

After the anchor restoration (after the block offset restoration loop), add a post-pass that lays out non-frozen upstream nodes that were unreachable during `place_subtree`.

In `layout_upstream`, after the freeze block anchor restoration block, add:

```python
# --- Post-pass: position non-frozen nodes upstream of freeze blocks ---
# place_subtree could not traverse through excluded freeze block members,
# so non-frozen nodes connected through them were never repositioned.
# Collect these nodes and run a secondary layout pass above the anchored block.
if freeze_group_map:
    for group_uuid, block_members in freeze_group_map.items():
        block_member_ids = {id(m) for m in block_members}
        # Find non-frozen upstream nodes reachable from block members
        upstream_of_block = []
        visited_upstream = set()
        queue = []
        for member in block_members:
            for inp in get_inputs(member):
                if inp is not None and id(inp) not in block_member_ids:
                    queue.append(inp)
        while queue:
            candidate = queue.pop()
            if id(candidate) in visited_upstream or id(candidate) in block_member_ids:
                continue
            if id(candidate) in freeze_excluded_ids:
                continue
            visited_upstream.add(id(candidate))
            upstream_of_block.append(candidate)
            for inp in get_inputs(candidate):
                if inp is not None:
                    queue.append(inp)
        if not upstream_of_block:
            continue
        # Position upstream nodes above the anchored block.
        # Find the topmost Y of the anchored block (min ypos of all members).
        block_min_y = min(m.ypos() for m in block_members)
        # Find the most-upstream entry point into the block (the block member
        # that has a non-frozen input — this is the connection point).
        # Use the block root's x position as the anchor column.
        block_root = freeze_block_roots[group_uuid]
        # Build a subtree filter for just the upstream nodes.
        upstream_filter = set(upstream_of_block)
        # Find the entry node: the upstream node directly connected to a block member.
        entry_nodes = []
        for member in block_members:
            for inp in get_inputs(member):
                if inp is not None and id(inp) in visited_upstream:
                    entry_nodes.append((inp, member))
        if not entry_nodes:
            continue
        # Position each entry subtree above the block.
        # Use the same layout parameters as the main pass.
        for entry_node, connecting_member in entry_nodes:
            entry_memo = {}
            compute_dims(
                entry_node, entry_memo, snap_threshold, node_count,
                node_filter=upstream_filter,
                scheme_multiplier=root_scheme_multiplier,
                per_node_h_scale=per_node_h_scale,
                per_node_v_scale=per_node_v_scale,
            )
            # Place the entry node so its bottom edge sits above the block's top edge
            # with the standard vertical gap.
            entry_y = block_min_y - vertical_gap_between(entry_node, connecting_member, snap_threshold) - entry_node.screenHeight()
            place_subtree(
                entry_node, connecting_member.xpos(), entry_y,
                entry_memo, snap_threshold, node_count,
                node_filter=upstream_filter,
                scheme_multiplier=root_scheme_multiplier,
                per_node_h_scale=per_node_h_scale,
                per_node_v_scale=per_node_v_scale,
            )
```

Apply the same pattern in `layout_selected` after its anchor restoration block. In `layout_selected`, the upstream filter should only include nodes that are also in the original expanded selection (they must be part of the layout scope).
