<objective>
Implement rigid block positioning (FRZE-06) and freeze-aware push-away (FRZE-07). After
Plan 01 provides freeze group detection, this plan makes the layout engine treat each
freeze block as a rigid unit: the root node is placed by place_subtree while other block
members are excluded and repositioned by their relative offsets. Push-away translates
entire blocks as a unit. Also fixes the folded Group View todo for Dot creation calls.

Purpose: Without rigid positioning, frozen nodes would be independently repositioned by
place_subtree, breaking the block. Without freeze-aware push, individual block members
would be pushed at different deltas, shattering the block.

Output: Rigid positioning logic in layout_upstream and layout_selected, freeze-aware
push_nodes_to_make_room, Group View fix for Dot creation, tests for all behaviors.
</objective>
