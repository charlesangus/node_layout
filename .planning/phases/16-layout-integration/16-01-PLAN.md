**Step 1: Create `_find_freeze_block_root(block_members)` helper (around line 1393, before layout_upstream):**

```python
def _find_freeze_block_root(block_members):
    """Return the most downstream node in *block_members*.

    The root is the block member that no other block member takes as an input.
    If multiple qualify (disconnected block), pick by max ypos() as tiebreaker
    (highest Y value = lowest on screen = most downstream in Nuke DAG).
    """
    member_id_set = {id(n) for n in block_members}
    candidates = []
    for member in block_members:
        is_input_to_another_member = False
        for other in block_members:
            if id(other) == id(member):
                continue
            for slot in range(other.inputs()):
                if other.input(slot) is not None and id(other.input(slot)) == id(member):
                    is_input_to_another_member = True
                    break
            if is_input_to_another_member:
                break
        if not is_input_to_another_member:
            candidates.append(member)
    if not candidates:
        return block_members[0]  # fallback
    # Tiebreak: most downstream = highest ypos (positive Y is down in Nuke DAG)
    return max(candidates, key=lambda n: n.ypos())
```

**Step 2: Create `_detect_freeze_groups(scope_nodes)` helper:**

This function:
1. Scans all `scope_nodes` and reads `node_layout_state.read_freeze_group(node)` for each.
2. Builds `freeze_group_map: dict[str, list[Node]]` (UUID -> members) and `node_freeze_uuid: dict[int, str]` (id(node) -> UUID).
3. Runs auto-join: For each non-frozen node in scope, checks if it has both a frozen ancestor AND a frozen descendant in the same group. Uses a downstream map built from `get_inputs()` for efficient descendant lookup within scope.
   - Build `downstream_map: dict[int, list[Node]]` where `downstream_map[id(input_node)]` = list of nodes in scope that take `input_node` as an input.
   - For each non-frozen node, BFS upward through `get_inputs()` (within scope) to find frozen ancestor UUIDs.
   - BFS downward through `downstream_map` to find frozen descendant UUIDs.
   - If intersection of ancestor UUIDs and descendant UUIDs is non-empty:
     - If exactly one UUID: node joins that group. Add to `node_freeze_uuid` and `freeze_group_map`. Call `node_layout_state.write_freeze_group(node, uuid)`.
     - If two or more UUIDs: merge all groups. Generate `merged_uuid = str(uuid.uuid4())`. Collect all members of all involved groups + the bridging node. Call `node_layout_state.write_freeze_group(n, merged_uuid)` for each. Update both maps (remove old UUIDs, add merged_uuid).
4. Returns `(freeze_group_map, node_freeze_uuid)`.

Important: The auto-join loop must be iterative — after joining/merging, re-check remaining non-frozen nodes since a newly joined node may enable further joins. Use a `changed = True` while loop or process nodes in topological order (upstream-first).

**Step 3: Create `_expand_scope_for_freeze_groups(selected_nodes, current_group)` helper:**

This function handles the "partial selection expands scope silently" requirement:
1. Scan `selected_nodes` for any that have a `freeze_group` UUID.
2. Collect all UUIDs found.
3. For each UUID, scan ALL nodes in `current_group` (via `current_group.nodes()` if not None, else `nuke.allNodes()`) to find all members of that group.
4. Return the union of `selected_nodes` and all discovered group members.
5. The extra (non-selected) members will be repositioned via relative-offset mechanism (FRZE-06, Plan 02), not via place_subtree.
