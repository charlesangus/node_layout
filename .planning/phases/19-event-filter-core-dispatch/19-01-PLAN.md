# Plan 19-01: Implement LeaderKeyFilter and arm() entry point

## Objective

Create `node_layout_leader.py` containing the `LeaderKeyFilter` QObject event filter and `arm()` module entry point. This module is the leader mode state machine: `arm()` installs the filter on QApplication, intercepts keypresses for V/Z/F/C single-shot dispatch, cancels on unrecognized input or mouse click, then removes itself via `_disarm()`.

## Context

Key decisions from 19-CONTEXT.md and 19-RESEARCH.md:

- **D-01/D-02:** Ephemeral install/uninstall with module-level singleton reused across cycles
- **D-04:** Double-arm guard via `_leader_active` bool check at top of `arm()`
- **D-05:** Set `_leader_active = True` and `installEventFilter` BEFORE scheduling overlay timer
- **D-06/D-07:** Use a named `QTimer` instance (NOT `QTimer.singleShot` static) so the timer can be cancelled in `_disarm()`. Read delay from `node_layout_prefs.prefs_singleton.get("hint_popup_delay_ms")`
- **D-08:** Discover DAG widget at arm() time via `QApplication.focusWidget()` parent chain walk; pass as parent to overlay
- **D-09:** arm() has NO focus guard itself; LEAD-04 is satisfied by Phase 21's `shortcutContext=2`
- **D-10/D-11/D-12:** Dispatch table for V (context-aware), Z (horizontal), F (freeze toggle), C (unfreeze)
- **D-13:** Unrecognized key: `_disarm()`, return `True` (consume)
- **D-14:** MouseButtonPress: `_disarm()`, return `False` (pass click through)
- **D-15:** When not active, `eventFilter()` returns `False` unconditionally
- **D-16:** Auto-repeat guard: `isAutoRepeat()` returns `True` to consume
- **D-18:** PySide6 enum syntax required: `Qt.Key.Key_V`, `QEvent.Type.KeyPress`, `QEvent.Type.MouseButtonPress`
- **Critical timer constraint:** `QTimer.singleShot(ms, cb)` returns None and CANNOT be cancelled. Must use named `QTimer` instance with `setSingleShot(True)` + `start(ms)` + `stop()`.

### Task 2: Implement LeaderKeyFilter class with eventFilter

**Files:** `/workspace/node_layout_leader.py`

**Action:**

Define `LeaderKeyFilter(QObject)` class with:

1. `__init__(self, parent=None)`:
   - Call `super().__init__(parent)`
   - Create the overlay timer as an instance attribute: `self._overlay_timer = QTimer()`, call `self._overlay_timer.setSingleShot(True)`. Do NOT connect timeout here -- connection happens in `arm()` when the overlay exists.
   - Store the timer reference in the module global `_overlay_timer` as well (so `_disarm()` can access it).

2. `eventFilter(self, watched, event) -> bool`:
   - First line: `global _leader_active` then `if not _leader_active: return False` (D-15)
   - Get `event_type = event.type()`
   - If `event_type == QEvent.Type.KeyPress`:
     - If `event.isAutoRepeat()`: return `True` (D-16 -- consume auto-repeat)
     - Get `key = event.key()`
     - Look up key in dispatch table dict `_DISPATCH_TABLE`
     - If found: call `_disarm()`, then call the dispatch function, return `True`
     - If not found: call `_disarm()`, return `True` (D-13 -- consume unrecognized key)
   - If `event_type == QEvent.Type.MouseButtonPress`:
     - Call `_disarm()`, return `False` (D-14 -- pass click through to Nuke)
   - Default: return `False`

### Task 3: Implement dispatch table and dispatch helper functions

**Files:** `/workspace/node_layout_leader.py`

**Action:**

1. Define `_dispatch_layout()` function:
   - `import nuke` and `import node_layout` (inline imports)
   - `selected_nodes = nuke.selectedNodes()`
   - If `len(selected_nodes) == 1`: call `node_layout.layout_upstream()`
   - Elif `len(selected_nodes) >= 2`: call `node_layout.layout_selected()`
   - (0 selected: no-op, which is correct -- nothing to layout)

2. Define `_dispatch_freeze_toggle()` function:
   - `import nuke`, `import node_layout`, `from node_layout_state import read_freeze_group` (inline imports)
   - `selected_nodes = nuke.selectedNodes()`
   - If `not selected_nodes`: return (no-op)
   - Check: `all_frozen = all(read_freeze_group(node) is not None for node in selected_nodes)`
   - If `all_frozen`: call `node_layout.unfreeze_selected()`
   - Else: call `node_layout.freeze_selected()` (D-11: any unfrozen means freeze all)

3. Define the dispatch table as a module-level dict (placed after the helper functions):
   ```python
   _DISPATCH_TABLE = {
       Qt.Key.Key_V: _dispatch_layout,
       Qt.Key.Key_Z: lambda: __import__('node_layout').layout_selected_horizontal(),
       Qt.Key.Key_F: _dispatch_freeze_toggle,
       Qt.Key.Key_C: lambda: __import__('node_layout').unfreeze_selected(),
   }
   ```
   Note: For Z and C, the simplest approach is a small wrapper function with an inline import, same pattern as V and F. Do NOT use `__import__` lambdas -- use named helper functions or simple wrappers with inline imports for clarity. For example:
   ```python
   def _dispatch_horizontal_layout():
       import node_layout
       node_layout.layout_selected_horizontal()

   def _dispatch_clear_freeze():
       import node_layout
       node_layout.unfreeze_selected()
   ```

   The dispatch table then becomes:
   ```python
   _DISPATCH_TABLE = {
       Qt.Key.Key_V: _dispatch_layout,
       Qt.Key.Key_Z: _dispatch_horizontal_layout,
       Qt.Key.Key_F: _dispatch_freeze_toggle,
       Qt.Key.Key_C: _dispatch_clear_freeze,
   }
   ```

### Task 4: Implement arm() and _disarm() module functions

**Files:** `/workspace/node_layout_leader.py`

**Action:**

1. Define `_find_dag_widget()` helper:
   - Call `focus_widget = QApplication.instance().focusWidget()`
   - If `focus_widget is None`: return `None`
   - Walk the parent chain: `widget = focus_widget`, then `while widget is not None`, check if it's a suitable DAG parent (any QWidget will do as a parent -- the overlay centering handles the rest), return `focus_widget.window()` or just walk up to find the top-level DAG container. Simplest correct approach: walk `parentWidget()` until reaching `None`, return the last non-None widget (the top-level widget containing the DAG).
   - Actually, the simplest correct approach per D-08: just return `focus_widget` itself or its immediate parent. The overlay centers itself over whatever parent it receives. Return `focus_widget` as the parent -- `LeaderKeyOverlay.show()` maps to global coordinates from the parent's rect, so the focus widget's parent (which is the DAG panel area) is ideal. Use: walk up until parent is `None`, return the widget just before that (the top-level window-like container).

2. Define `arm()`:
   - `global _leader_active, _filter, _overlay, _overlay_timer`
   - If `_leader_active`: return immediately (D-04 double-arm guard)
   - If `_filter is None`: create `_filter = LeaderKeyFilter()` and set `_overlay_timer = _filter._overlay_timer`
   - Find DAG widget: `dag_widget = _find_dag_widget()`
   - If `_overlay is None`: create `_overlay = LeaderKeyOverlay(parent=dag_widget)` (import `from node_layout_overlay import LeaderKeyOverlay` inline)
   - Else if `dag_widget is not None`: call `_overlay.setParent(dag_widget)` to re-parent for current DAG panel
   - Set `_leader_active = True` (D-05: BEFORE installing filter and scheduling timer)
   - Install filter: `QApplication.instance().installEventFilter(_filter)`
   - Schedule overlay: disconnect any prior timeout connection (to avoid duplicate connections), then connect `_overlay_timer.timeout.connect(_overlay.show)` and start with delay: `delay_ms = node_layout_prefs.prefs_singleton.get("hint_popup_delay_ms")` then `_overlay_timer.start(delay_ms)` (import `node_layout_prefs` inline)

   For the timer connection, the cleanest pattern to avoid duplicate connections:
   ```python
   try:
       _overlay_timer.timeout.disconnect()
   except RuntimeError:
       pass  # No connections to disconnect
   _overlay_timer.timeout.connect(_overlay.show)
   _overlay_timer.start(delay_ms)
   ```

3. Define `_disarm()`:
   - `global _leader_active`
   - If `not _leader_active`: return (guard against double-disarm)
   - Set `_leader_active = False`
   - `QApplication.instance().removeEventFilter(_filter)` (safe from within eventFilter per Qt docs)
   - `_overlay_timer.stop()` (cancel pending show; no-op if not running)
   - If `_overlay is not None`: `_overlay.hide()`
