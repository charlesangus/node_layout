**Class `LeaderKeyOverlay(QWidget)`:**

`__init__(self, parent=None)`:
1. `super().__init__(parent)`
2. `self.setWindowFlags(Qt.WindowType.Tool | Qt.WindowType.FramelessWindowHint)`
3. `self.setAttribute(Qt.WidgetAttribute.WA_ShowWithoutActivating)`
4. `self.setAttribute(Qt.WidgetAttribute.WA_TranslucentBackground)`
5. Call `self._build_ui()`
6. Call `self.adjustSize()` — ensures rect() has real dimensions for centering in show().

`_build_ui(self)`:
1. Create a `QVBoxLayout` as main layout with some padding (e.g., `setContentsMargins(16, 12, 16, 16)`).
2. Create "LEADER KEY" title `QLabel`:
   - Text: `"LEADER KEY"`
   - Font: bold, size ~11pt
   - Color: white/light gray via stylesheet `color: #cccccc;`
   - Alignment: `Qt.AlignmentFlag.AlignCenter`
   - Add to main layout.
3. Create a `QGridLayout` for the key grid:
   - Spacing: ~8px between cells via `setSpacing(8)`
   - `setColumnMinimumWidth(1, 64)` — gives the empty X column real width
4. For each `(key_letter, action_label, row, col)` in `_KEY_LAYOUT`, call `self._make_key_cell(key_letter, action_label)` and add the returned widget to grid at `(row, col)`.
5. Add grid layout to main layout.
6. `self.setLayout(main_layout)`

`_make_key_cell(self, key_letter, action_label)`:
1. Create a container `QWidget`.
2. Create a `QVBoxLayout` for the container with tight margins.
3. Determine badge color: `_CHAINING_KEY_COLOR if key_letter in CHAINING_KEYS else _SINGLE_SHOT_KEY_COLOR`.
4. Create key badge `QLabel`:
   - Text: the `key_letter`
   - Font: bold monospace (e.g., `QFont("monospace", 14, QFont.Weight.Bold)`) — per D-02
   - Alignment: centered
   - Style: use stylesheet with `background-color` from the badge color's RGB, `color: #111111`, `border-radius: 4px`, `padding: 4px 12px`, `min-width: 28px`
5. Create action label `QLabel`:
   - Text: the `action_label`
   - Font: size ~8pt
   - Color: gray via stylesheet `color: #999999;` — per D-02
   - Alignment: centered
6. Add key badge and action label to container layout.
7. Return the container widget.

`paintEvent(self, event)`:
1. `painter = QPainter(self)`
2. `painter.setRenderHint(QPainter.RenderHint.Antialiasing)`
3. `painter.setBrush(QColor(20, 20, 20, 180))` — ~70% opacity dark background per D-01
4. `painter.setPen(QColor(180, 180, 180, 100))` — subtle border
5. `painter.drawRoundedRect(self.rect(), 8, 8)` — rounded corners

`show(self)`:
1. Call `super().show()` first (so the native window exists).
2. `parent = self.parentWidget()`
3. If parent is not None:
   - `global_center = parent.mapToGlobal(parent.rect().center())`
   - `self.move(global_center - self.rect().center())`
   Per D-08: centered over parent on each show() call. `move()` is called AFTER `super().show()` per research Pitfall 4 guidance.

No other methods needed. `hide()` is inherited from QWidget — per D-14, it fully dismisses the widget.
